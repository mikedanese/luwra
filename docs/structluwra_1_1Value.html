Detailed Description

template<typename T>
struct luwra::Value< T >

Wrapper for a stack value

Note
This generic version should not be used since it does not implement anything. It is not always obvious whether you have used this generic instance or not. Therefore this class produces an error message when it is being used. Look for Parameter to Value is not supported in your compiler output.
